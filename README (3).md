# Простий калькулятор
Консольний калькулятор для виконання базових арифметичних операцій.  
Навчальний проєкт для вивчення Git, GitHub та Python.

## Опис
Цей проєкт представляє собою простий калькулятор з текстовим інтерфейсом, який дозволяє виконувати основні математичні операції в інтерактивному режимі. Програма приймає операцію та два числа, обробляє помилки (як-от ділення на нуль) і виводить результат.

## Функціональність
- **Додавання:** Додає два числа
- **Віднімання:** Віднімає друге число від першого
- **Множення:** Множить два числа
- **Ділення:** Ділить перше число на друге з перевіркою ділення на нуль
- **Степінь:** Підносить перше число до степеня другого
- **Модуль:** Обчислює остачу від ділення першого числа на друге

## Вимоги
- **Python:** 3.7 або вище
- **ОС:** Windows, macOS або Linux
- **Залежності:** Відсутні (стандартна бібліотека)
